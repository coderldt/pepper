> A Vue.js 2.0 UI Toolkit for Web.

## 组件贡献指南
```bash
yarn
yarn new [组件名] [组件中文名] [组件分组名]
yarn dev
```
> 参数组件名为必填参数，组件中文名、组件分组名为可选参数(分组名不填默认在 Others 下, 添加分组名参数若配置内无已存在分组则会新建分组)

## COMMIT
```bash
yarn commit
```
